PacENeT

A user portal app for ISP service

Tech Stack:

  • Android: Kotlin, Retrofit, ViewPager2, Architecture components, FCM, Dagger 2, Android navigation, Kotlin coroutines
  • iOS: SwiftUI, Combine, URLSession, ios-charts (github)
Platform: iOS, Android

Key Features:

  • Making instant changes to current bandwidth
  • Data usage report with chart
  • All payments from app
  • Changing current data package subscription
  • Daily, monthly, yearly reports
  • Chat with support team
