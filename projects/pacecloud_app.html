PacEClouD

A control panel app for customers of cloud services

Tech Stack:

  • Android: Kotlin, Retrofit, Architecture components, Dagger 2, Android navigation, Kotlin coroutines
  • iOS: SwiftUI, Combine, URLSession, ios-charts (github)
Platform: iOS, Android

Key Features:

  • On, off, restart VM PCs remotely
  • Cloud storage usage report with chart
  • All payments from app
  • Cloud storage usage report...
  • Active-inactive VM PC list
  • Chat with support team
